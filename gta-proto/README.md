# GTA-proto

Мини-прототип в духе GTA на Three.js (vanilla JS + Vite). Single page, запускается в браузере.

## Что внутри

- Кубический город 400×400 с грид-улицами, тротуарами и зданиями
- Пеший персонаж от 3-го лица (WASD, мышь-камера, Shift-бег, Space-прыжок)
- Угоняемая машина (F — сесть/выйти, физика-lite: газ, тормоз, ручник, поворот)
- ~60 NPC-пешеходов с случайным блужданием
- Стрельба (ЛКМ) с трассерами и хит-детектом (ray-sphere + LOS через здания)
- Розыск (0–5★): растёт за убийства пешеходов/копов, спадает со временем
- Полиция: спавнится пропорционально розыску, догоняет и стреляет
- Магазин оружия: красный дом, E — купить пистолет ($250) или патроны ($100/60)
- Миссия-доставка: жёлтый маркер → зелёный маркер, награда $800
- HUD: HP / патроны / деньги / звёзды / мини-карта / подсказки
- Простой WebAudio SFX (выстрел, сирена, касса, попадание)

## Запуск

```bash
cd gta-proto
npm install
npm run dev       # Vite dev-сервер на http://localhost:5173
# или
npm run build && npm run preview
```

## Управление

| Клавиша | Действие |
|---------|----------|
| WASD / стрелки | Движение (пешком и за рулём) |
| Мышь | Камера |
| ЛКМ | Стрелять |
| Shift | Бег / ручник в машине |
| Space | Прыжок / тормоз в машине |
| F | Сесть / выйти из ближайшей машины |
| E | Взаимодействие (магазин / миссия) |
| R | Перезарядка |
| Esc | Освободить курсор |

## Архитектура

```
src/
  main.js       — рендерер, главный цикл, шутинг, миссия, камера
  world.js      — генерация города, коллайдеры (AABB), shopZone/mission маркеры
  player.js     — пешеход от 1-го лица + кукольная анимация
  car.js        — машина (простая arcade-физика)
  npc.js        — пешеходы с wander-AI
  police.js     — копы + PoliceSystem (звёзды, спавн, чейз)
  input.js      — клавиатура/мышь/pointer lock
  hud.js        — DOM HUD, баннеры, промпты
  minimap.js    — 2D мини-карта через <canvas>
  audio.js     — WebAudio SFX без файлов
```

Физика — простая AABB-коллизия с зданиями + сдвиг по XZ. Пули — ray-sphere против сущностей + sampled segment через здания. Всё работает полностью на клиенте, без бэкенда.
